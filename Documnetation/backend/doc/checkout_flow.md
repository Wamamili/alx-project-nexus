Checkout Process Overview

This document explains the steps users follow to complete a purchase on ByteMtaani.

Step 1: Cart Validation

Validate cart items

Check product availability

Recheck updated prices if promotions apply

Step 2: Delivery Details

User selects shipping address

Delivery fee is calculated based on location

Step 3: Order Draft Creation

Create order with status pending

Generate order items

Store totals and metadata

Step 4: Payment Start

Produce transaction reference

Initialize payment on provider

Redirect user or open payment modal

Step 5: Confirmation

Provider sends verification data

Backend confirms status

Update order status to paid or failed

Notify user

Step 6: Fulfillment Dispatch

Staff receives order

Fulfillment pipeline begins
